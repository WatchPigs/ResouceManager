Please install graphviz by homebrew, with command in terminal:
brew install graphviz

If you don't have homebrew, install it first, with command in terminal:
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

Please run it with superuser, it means with command in terminal: 
cd xxx/xxx/(currentPath)
sudo ./ResouceManager
